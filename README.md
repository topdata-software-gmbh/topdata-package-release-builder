# Shopware Plugin Builder

A tool for building and packaging Shopware 6 plugins.

## Installation

```bash
uv pip install -e .
```

## Development Setup

1. Clone the repository
2. Create a virtual environment:
   ```bash
   uv venv
   source .venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   uv pip install -e ".[dev]"
   ```

## Usage

```bash
sw-build --help
```
